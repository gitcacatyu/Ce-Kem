Aula 4 – Cartão com CSS


<style>
  .card {
    background: white;
    padding: 20px;
    border-radius: 10px;
    width: 250px;
    margin: auto;
    text-align: center;
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
  }
  .card img {
    width: 100px;
    border-radius: 50%;
  }
</style>

<div class="card">
  <img src="https://via.placeholder.com/100" alt="Foto">
  <h2>Kaká</h2>
  <p>Estudante de programação</p>
</div>

Aula 5 – Responsividade


<style>
  .caixa {
    width: 80%;
    background: lightblue;
    padding: 20px;
    margin: auto;
  }
  @media (max-width: 600px) {
    .caixa {
      background: lightcoral;
      width: 100%;
    }
  }
</style>

<div class="caixa">Caixa responsiva</div>

Aula 6 – Animações CSS


<style>
  .bola {
    width: 50px;
    height: 50px;
    background: red;
    border-radius: 50%;
    animation: girar 2s linear infinite;
  }
  @keyframes girar {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
  }
</style>

<div class="bola"></div>

Aula 7 – Botão com Gradiente


<style>
  .btn-grad {
    background: linear-gradient(90deg, orange, blue);
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    transition: 0.3s;
  }
  .btn-grad:hover {
    background: linear-gradient(90deg, blue, orange);
  }
</style>

<button class="btn-grad">Clique Aqui</button>
